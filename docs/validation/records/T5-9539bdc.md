## Integration Flow Analysis

The connected flow was traced end-to-end:

1. **Launch**: `project.godot` → `main_scene = main.tscn` → GameSession + FlowController + UIRoot/SelectionOverlay + UIRoot/DayHUD initialize
2. **Troop Selection**: FlowController._ready() → shows troop_selection UI → user clicks → troop_selected.emit()
3. **Battle Entry**: FlowController._on_troop_selected() → _start_battle() → battle_root scene instantiated → spawn_player_units() → player units cleared + spawned at INFANTRY_POSITIONS or CAVALRY_POSITION
4. **Unit Movement**: SelectionOverlay handles left-click selection (box or single) → game_session.set_selected_squads() → right-click issues move command via battle_root._issue_move_command() → formation movement to target
5. **Combat**: battle_root._physics_process() → _update_combat_targets() every 0.25s → auto-targeting → squad._physics_process() → navigation toward enemy → melee range check → ATTACK_INTERVAL + take_damage() → queue_free() on death
